# AOS

Analysis Of Sales
============

기존의 POS시스템의 단점인 매출의 시각화와 매출 분석을 위한 프로그램<br>
-------------
라이브러리 활용 능력을 높이기위해 다양한 라이브러리 사용<br>
-------------



## 사용 라이브러리
### 1. JDOM <br>
- 기상정보 활용을 위해 사용한 라이브러리
- 기상청에서 제공해주는 xml파일을 파싱하여 실시간 기상정보를 가져옴

### 2. Jxbrowser
- 실시간 판매현황을 시각화하기 위해 사용한 라이브러리
- 차트론 구글 차트를 이용
